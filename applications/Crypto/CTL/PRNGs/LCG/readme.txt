RVC-CAL Implementation of Linear Congruential Generator (LCG)

**********************************************************

Junaid Jameel Ahmad, Shujun Li @ Uni-Konstanz

19 August, 2010

**********************************************************

There are two versions of LCG:

LCG32: Supports initialization parameters and generated output numbers to be upto 32-bits.
LCG64: Supports initialization parameters and generated output numbers to be upto 64-bits. 
 Note: Due to un-availibity of Big Integers manipulation, the output stream of this FU has not been verified yet.
